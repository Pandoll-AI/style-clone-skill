# 측정 명세 v2 — 특징 정의와 수식

표기: 문장열 s_1..s_n, 내용어 집합 C(s_i), 어절 수 |s_i|, 임베딩 e(s_i).

## 구현 완료 (T0, discourse.py)

| 특징 | 정의 | 층위 |
|---|---|---|
| implicit_ratio | 문두 연결 표지가 없는 문장 비율 | L4 |
| connective_dist | 문두 표지의 6유형(인과·역접·부연·예시·전환·첨가) 분포 | L5 |
| top_transitions | 연결 유형 bigram 전이의 상위 5개 빈도 | L5 |
| jump_mean/std/p90 | j_i = 1 − |C(s_{i−1})∩C(s_i)| / min(|C(s_{i−1})|,|C(s_i)|) 의 분포 | L5 |
| flight_return_rate | j_i>0.8인 도약 중, C(s_i)가 s_{i−2}∪s_{i−3}과 20% 이상 겹치는 비율 | L5 |
| len_cv, len_autocorr, len_alternation | 길이 시계열의 CV, lag-1 자기상관, 증감 부호 교대율 | L6 |
| jump_autocorr | 도약 크기 시계열의 lag-1 자기상관 (전개 템포의 관성) | L6 |
| new_info_by_quartile | 문장별 신규 내용어 비율의 사분위 평균 | L6/L8 |
| question/len_by_quartile | 위치 조건부 프로파일 | L8 |
| echo_head_tail | 첫 1할과 끝 1할 내용어의 중첩/min | L8 |
| mtld_lite | 50어 창 이동 TTR 평균 | L9 |
| burstiness_fano | 상위 내용어 10개의 문장 등장 지시열 Fano factor 평균 | L9 |
| len_gini | 문장 길이 Gini 계수 | L9 |

## 구현 예정 (T1–T4, embedding_features_spec.py 에 인터페이스)

| 특징 | 정의 | 도구 |
|---|---|---|
| semantic_jump | 1 − cos(e(s_{i−1}), e(s_i)) 분포 + 꼬리 지수 | T2 |
| topic_drift_curve | cos(e(s_i), e(s_{i+k}))의 k에 대한 감쇠율 | T2 |
| entailment_gap | 1 − P_NLI(s_{i−1} ⇒ s_i 문맥 연결) — '추론 보폭'의 정밀판 | T4 |
| surprisal_mean/cv | 어절 surprisal의 평균(정보 밀도)·CV(템포 일정함) | T3 |
| surprisal_spectrum | 궤적의 저주파 성분 비율(리듬 주기성) | T3 |
| idea_density | 어절당 용언+수식언 수 | T1 |
| syntax_depth | 관형절 중첩 깊이 분포, '것' 명사화 밀도 | T1 |
| subject_ellipsis | 주어 없는 문장 비율 | T1 |
| hedge_booster_ratio | 완화 표지 대 단정 표지 비율 | T0/T1 사전 |
| entity_grid_transition | 개체의 S/O/X 역할 전이 분포 (일관성) | T1+T4 |
| arc_shape | 사분위별 감정가·긴장도 곡선의 형태 계수 | T4 |

## 채점 규칙 v2 (v1에서 변경)

1. v1과 동일: 수용 z(작가 청크 분산 정규화) / 판별·재순위 d(절대 거리) 분리.
2. **산포 비대칭 벌점**: 산포 계열(F_*)은 |z|가 아니라 방향 부호를 유지한다. 후보 산포가 작가 하한보다 낮으면(과평탄) 벌점 α=1.5배 가중, 높으면 표준 가중. 근거: LLM 오류 모드가 평탄화 쪽으로 체계적으로 치우침(문헌 실증 + 본 프로젝트 C2 실측).
3. **층위별 판별기 분리**: 재순위화 = L1+L4+L6 (계산 빠름). 수용 판정 = 전 층. 외부 검증 = 문자 bigram + 블라인드 LLM + (신규) L5 지표. 최적화에 쓴 층은 해당 실험의 평가에서 제외.
4. 위치 조건부화: 모든 L1 분포를 사분위별로도 저장(기승전결 배분). 표본이 작으면 전역 분포로 백오프.

## 알려진 한계와 편향

- T0 도약 프록시는 어휘 중첩 기반이라 동의어 치환에 취약(임베딩 도입 시 해소). jump_p90은 짧은 문장에서 포화(=1.0)하므로 p75 병용 권장.
- 문두 표지 분류는 표층 매칭이라 "그리고"로 시작하지 않는 첨가 관계를 놓침 — implicit_ratio는 상한 추정치.
- 사분위 프로파일은 n<40 문장에서 불안정. 최소 표본 크기를 특징별로 기록하고 CI 넓으면 채점 제외(v1 원칙 유지).
