# 문체 차원 분류체계 (Style Dimension Taxonomy) v2

전제: "문체"는 단일층이 아니라 최소 9개 층위의 확률적 습관의 합성이다. v1은 L1만 측정했다. 각 층마다 (a) 정의, (b) 현상 사례, (c) 측정 프록시, (d) 필요 도구 티어를 명시한다.

도구 티어: T0 = 순수 파이썬(구현 완료 가능) · T1 = 형태소 분석기(kiwipiepy) · T2 = 문장 임베딩(KoSimCSE, BGE-M3) · T3 = 언어모델 surprisal(Polyglot-Ko 등) · T4 = NLI/특화 모델(klue-roberta-nli, argument mining)

## L1. 표층·형태 (v1 구현 완료)

종결어미·연결어미·조사·부사 분포, 문장 길이 분포, 문장부호 습관, 첩어 밀도, 금지 목록. — `stylometry.py`. T0(현재)→T1(정밀화).

## L2. 통사 구조

문장을 어떤 뼈대로 짓는가. 내포절 깊이(안긴문장 중첩), 관형절 밀도, 명사화 밀도(~것/~음/~기), 피동/사동 비율, 주어 생략률(한국어 특유의 강한 개인차), 병렬 구문율, 어순 도치 빈도.
- 프록시: 관형형 어미(-은/-는/-던) 밀도, '것' 밀도, 어절당 용언 수. T1에서 의존구문 없이도 태그 시퀀스로 근사 가능.

## L3. 어휘·의미 선택

같은 뜻에서 어떤 단어를 뽑는가. 어종 비율(고유어/한자어/외래어), 추상도(추상명사 비율), 감각어·구체어 밀도, 어휘 다양성(MTLD), 레지스터(구어/문어 표지), 상투어 회피 성향.
- 프록시: T1 + 어종 사전. 추상도는 접미(-성/-화/-도/-론) 밀도로 T0 근사.

## L4. 결속 (Cohesion)

문장을 무엇으로 붙이는가. 명시적 결속(접속 표지, 지시어 '이런/그') vs 암시적 결속(어휘 반복, 생략). **무표지율이 1차 지표**: 논리 표지를 깔면서 가는 필자 vs 독자에게 접속을 맡기는 필자.
- 프록시: 문두 연결 표지 분류 + 무표지율 (T0, `discourse.py` A). 실측: 소설가들 0.89, 대상 에세이 0.81, LLM 베이스라인 0.71 — LLM은 접속사를 과잉 사용한다.

## L5. 논리·추론 구조 (사용자 지적의 핵심)

- **추론 보폭(inferential step size)**: 인접 문장 사이에 독자가 메워야 하는 추론의 양. 좁으면 '꼼꼼한 전개', 넓은데 일관되면 '천재의 스킵', 넓은데 비일관이면 '단절'.
  프록시: T0 = 내용어 중첩 역수(`B_jump_mean`), T2 = 임베딩 코사인 거리, T4 = NLI entailment 확률의 역수(정밀).
- **사고의 비행(flight) vs 단절**: 큰 도약 후 앞 문맥으로 복귀하는가. `B_flight_return_rate` (T0). 실측: 에세이 0.16(도약 후 회귀하는 나선 구조) vs 소설 0.03~0.07(전진 서사).
- **연결 유형 전이 행렬**: 인과→역접→부연 같은 전개 패턴의 마르코프 습관 (T0).
- **논증 형태**: 주장-근거-예시 비율, 결론 위치(두괄/미괄), 자기반박 후 재긍정 습관 (T4 argument mining, 또는 LLM 라벨링).

## L6. 정보 흐름·템포 (사용자 지적: "전개 템포의 일정함")

- **surprisal 궤적**: 어절 단위 -log p의 시계열. UID 가설은 균일성을 예측하지만 실제 필자는 고유한 궤적 모양(피크 배치, 주기성)을 가진다 — 최근 연구는 정보 궤적의 조화(주기) 구조까지 확인. 평균 surprisal = 정보 밀도(글의 난도), 궤적 CV = 템포의 일정함, 자기상관/스펙트럼 = 리듬 주기 (T3).
- **아이디어 밀도**: 어절당 명제 수 (T1 근사: 용언+수식언 밀도).
- **신정보 도입률**: 문장별 신규 내용어 비율과 그 사분위 곡선 (T0, `D_new_info_by_quartile`). 실측: 에세이는 0.71→0.37로 선형 감소(수렴형), LLM은 말미까지 0.66(발산형).
- **문장 리듬**: 길이 시계열의 자기상관·교대율 (T0). 실측: 에세이 +0.20(완만한 파동) vs C2 -0.34(기계적 장단 교대) — v1이 못 잡은 결함.

## L7. 화용·태도·독자 관계

- 확신도 스펙트럼: hedge(~일지도, ~인 듯) vs booster(반드시, 분명) 비율 (T0 사전).
- 1인칭 노출도, 독자 호명(질문/청유/명령), 아이러니·자기비하·유머 표지.
- 인용 습관: 가상 대화 삽입 빈도와 기능(권위 인용 vs 극화).
- 프록시: T0 사전 기반 + T4(스탠스 분류).

## L8. 거시 구조 (기승전결)

- **위치 조건부 프로파일**: 모든 특징을 사분위 위치의 함수로 다시 측정. 기(신정보 급증·질문 제기), 승(전개), 전(역접 밀도 피크), 결(신정보 최소·회귀). 배분 비율 자체가 필자 서명 (T0, `E_*`).
- **수미상관(echo)**: 첫 1할과 끝 1할의 어휘 중첩. 실측: 에세이 0.29(강한 수미상관 — "AI는 평범함을 죽인다" 후렴) vs 소설 0.10.
- 문단 길이 분포, 소제목 리듬, 후렴 반복 구조.
- 정밀판: 위치별 감정가 곡선(narrative arc), RST 관계 트리의 모양 (T4).

## L9. 산포·평탄화 (메타 층 — 사용자 지적: "평탄화도 체크리스트에 넣어라")

각 층의 **분산 자체가 1급 특징**이다. 문헌은 instruction-tuned LLM이 어휘·내용 다양성을 체계적으로 줄여 서로 다른 필자의 글을 균질화함을 실증했다. 따라서:
- 측정: 문장 길이 CV·Gini, 도약 크기 분산, MTLD, 내용어 burstiness(Fano), 리듬 변동성 (T0, `F_*`).
- 채점 규칙: **산포는 '이상 편차'가 아니라 매칭 대상**. 후보의 산포 < 작가 산포 하한이면 '과평탄' 감점 (산포 초과와 비대칭으로 처리).
- 실증: 본 프로젝트 v1의 승자 C2가 실제로 과평탄이었다(MTLD 0.76 vs 작가 0.83, 길이 CV 0.34 vs 0.53). v1 지표만 최적화하면 평탄화 오버피팅이 발생한다는 사용자 가설이 데이터로 확인됨.

## 층위 간 관계

L1-L3은 국소(문장 내), L4-L6은 국지(문장 간), L7은 전역 태도, L8은 전역 구조, L9는 모든 층의 2차 모멘트. 재순위화는 L1+L4+L6(빠른 계산), 수용 판정은 전 층, 판별기는 층별로 분리 배치해 Goodhart 순환을 막는다.
